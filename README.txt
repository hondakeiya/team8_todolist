ToDoLIstの使い方

このアプリはpytonファイルです
tkinterというライブラリも使用しているのでインストールが必要です。
また、利用しているtkinterというライブラリのバグで
pyenv以下の環境では日本語入力ができないので利用する際は
注意してください

1、フォームに課題のタイトル、詳細、期限を入力
2、やることができたんだな？ボタンを押して課題を設定
3、完成した課題はクリックして完成を誇れ！お前は富士山だ！ボタンで削除

重要！！！！！！！！！！！！！！
やる気が減ったり落ち込んできたりした時には君ならできる！！ボタンで
松岡修造さんに励ましてもらえましょう。自己肯定感を上げてくれます。
